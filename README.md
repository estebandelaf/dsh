dsh
===

dsh (delaf's Shell) corresponde a una Shell de juguete, cuyo principal objetivo 
es implementar funcionalidades básicas para una interfaz de línea de comando o 
CLI.

No pretende ser, ni será, compatible con una Shell como bash o sh. Mucho menos 
implementará Shell Script.

Las características que podrá encontrar en esta shell son:

1. Ejecución de comandos interactivos.
2. Procesamiento por lotes de comandos (desde un archivo).
3. Tuberías (pipes).
4. Redireccionamiento de entrada y salida.

Los puntos 3 y 4 no están implementados, espero implementarlos en algún momento.
